When Invented?

It is thought that upper-class Victorians in England invented table tennis in the 1880s as a genteel, after-dinner alternative to lawn tennis, using whatever they could find as equipment.


Who Invented?

The earliest surviving action game of Tennis on a table is a set made by David Foster, patented in England in 1890 (No. 11037): Parlour Table Games, which included table versions of Lawn Tennis, Cricket and Football.
